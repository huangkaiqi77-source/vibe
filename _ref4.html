    随着人们生活水平的不断提高，人民群众的需要已经不再局限于物质满足，

    人们开始期盼有

    更好的教育、更满意的收入、更可靠的社会保障、更舒适的居住条件、更丰富的精神文化生活，

    影视行业因而开始蓬勃发展。

    影视剧市场现状

  • 各地域受众对电视剧类型偏好

     0

     0

    根据数据显示， “爱情”、“甜宠”仍是现今都市剧中的主要内容元素；为了使得都市爱情剧更具吸引力和话题度，制作方在剧集设定时往往会加入其他的元素，如“拯救”、“女性”、“成长”等，以便更好地触达各圈层的受众。

     

    都市剧数量在各类型电视剧中有着压倒性的领先优势，且在仍逐年上升。

    由于单一梦幻爱情式的偶像剧在市场上已逐渐被观众厌弃，现今的都市爱情剧开始将如女性、正能量、职业特点等多元元素融合，以此到达迎合不同受众口味的效果。

    当前都市剧主要有几种类型：一，以年轻受众为主的都市偶像剧；二，围绕时代发展、人民美好生活、职业题材等现实议题展开的正能量剧。

     

    在一般情况下，第二种类型的剧集在话题热度和获得好的市场口碑上往往高于第一类。

    0

    根据2022年全年数据显示，在电视剧头部市场用户一、二线城市与三线及以下城市中分布较为均衡，较于之前，呈一、二线城市受众减少、三线及以下城市人群增多，下沉市场影响力增加的趋势。

     

    一、二线城市的观众对于新鲜的内容接受程度比较高，而三线及以下城市的观众偏向保守，易于传播的多为已经开拓成熟的题材类型，如都市、古装、年代等。

    琼瑶爱情的落幕

  •  

    该剧在宣发时，着重强调了该剧聚焦一线基层消防战士和医护人员，通过对这两个职业线的细节刻画，展现了消防员与医护人员的艰辛与坚守的特点。 

     

    《我的人间烟火》以双向救赎的爱情、正能量为卖点，加以医生和消防员两个社会美誉度极高的职业设定，那么本应是热度、口碑双丰收的《我的人间烟火》，为何豆瓣评分却低至3.3？

    在《你是我的烟火》中，有这样一段剧情：女主角和男主角到灾区救援，在一处废墟中发现了一名被困的孕妇，通过手摸脉搏的判断，女主角发现孕妇已经死亡了，但是腹中的胎儿还有心跳，为了救治婴儿，女主角在废墟中进行剖宫产，最终成功取出了婴儿，所有人为此而欢呼。

     

    灾区救援剖宫产的剧情本应是该剧的“高光”场景，进一步凸显了医生以及消防员“逆行者”的伟大。

     

    然而，这在真正的医生们眼中就是无稽之谈，女主角不具备行医资格、没有抢救孕妇、在未经麻醉的情况下活生生将人剖开.....

     

    从关于该剧情片段的讨论中，“孕妇”、“医生”、“侮辱”等为高频词，体现人们对于编剧、剧情的bug的不满，对于该剧抹黑职业形象的愤怒。

  • 女性意识的觉醒

  • 0

    0

    《我的人间烟火》评分、风评跌入谷底，除了本身剧情在常理上有很多不合理之处，更多的还有当下女性对于“霸总”、“傻白甜”、“爱情至上”的影视剧的排斥。

     

     

    从《我的人间烟火》女主角许沁的相关热门话题上看，我们可以看出，引起网友热议的相关话题多为相关负面消息，传递出网友们对于该剧女主角的“爱情”不认可。

     

     

    这从本质上看，是女性意识的觉醒，当代女性更愿意追求“实现自我”、“奋斗成长”的“大女主”生活，而非“爱情大过一切”的过时观念。

    结语

    0

    随着人们精神物质财富的丰富，人们的观念向着更加先进的方向迈进，作为引导社会正能量、培育社会风气的精神文化作品更是要更新创作理念与思路，才能更好地顺应时代的发展。

     

    虽然近年来涌现出了不少交口称赞的精品好剧，不可否认，当今影视剧的创作上，尤其是以爱情类故事为主线的影视作品还有许多的不足，需要创作者们切实体会人民群众的生活，才能创作出“从群众中来，进到群众心里去”的好作品。
